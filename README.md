# ITPM Assignment 1 - Singlish to Sinhala Transliteration
**Student Registration Number:** IT23589322

## Description
This Playwright project automates 35 test scenarios to evaluate the accuracy and robustness of the Singlish-to-Sinhala transliteration system at swifttranslator.com.

## Setup Instructions
1. Ensure you have **Node.js** installed on your system.
2. Open the terminal in the project folder.
3. Install dependencies:
   ```bash
   npm install

4. To run this repo
   ```bash
   npx playwright test --project=chromium
